- Responsive design
- Animated sections

- Contact form

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- React.js
- GSAP (for animations)

## 📥 Installation

1. Clone this repository
```bash
git clone https://github.com/yourusername/Animated-Portfolio
```

2. Navigate to project directory
```bash
cd Animated-Portfolio
```

3. Install dependencies
```bash
npm install
```

4. Run the development server
```bash
npm start
```
